NEHRA CLASSES SNAKE GAME FOR CLI (Works on all fedora based Linux distros): 
---------------------------------------------------------------------------

Clone the repository:
# git clone https://github.com/nehraclasses/Linux-Software-Package-RPM-Files

Move to the directory containing the game RPM file.
# cd Linux-Software-Package-RPM-Files

List the game RPM file. (Optional)
# ls -lh nehra-classes-snake-1.0-1.el10.noarch.rpm

Install the game RPM file:
# rpm -ivh nehra-classes-snake-1.0-1.el10.noarch.rpm

Run the game:
# nehra-classes-snake

Uninstall (if needed)
# rpm -e nehra-classes-snake

NEHRA CLASSES RPM Package for a Login Animation (Works on all fedora based Linux distros): 
------------------------------------------------------------------------------------------

Clone the repository:
# git clone https://github.com/nehraclasses/Linux-Software-Package-RPM-Files

Move to the directory containing the game RPM file.
# cd Linux-Software-Package-RPM-Files

List the RPM file. (Optional)
# ls -lh nehra-login-anim-1.0-1.el10.noarch.rpm

Install the RPM package file:
# rpm -ivh nehra-login-anim-1.0-1.el10.noarch.rpm

Log out and log back in (or open a new terminal session).
You should see the animation:

🚀 Welcome to Nehra Classes 🚀

moving across the screen before clearing.

Uninstall (if needed):
# rpm -e nehra-login-anim
